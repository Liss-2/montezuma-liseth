Pregunta 1

¿Qué es Javascript y en qué se asemeja a Java?

R= JavaScript es un lenguaje de programación interpretado, dinámico y orientado a objetos que se utiliza principalmente para crear páginas web interactivas. Fue desarrollado por Netscape en 1995 y se ejecuta en el navegador del cliente (front-end), aunque también se puede usar en el servidor (back-end) con plataformas como Node.js. JavaScript es esencial para el desarrollo web moderno, permitiendo la manipulación del DOM (Document Object Model), validación de formularios, animaciones, entre otros.
